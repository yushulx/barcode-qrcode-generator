# Barcode & QR Code Generator

Generate 1D and 2D barcodes instantly from any webpage URL or custom text.

https://github.com/user-attachments/assets/8fc730b1-7ad8-4036-a96e-0c43ac37396f

## Features

- **One-Click QR Codes**: Convert current page URL to QR code instantly
- **13 Barcode Types**: Support for QR Code, EAN13, CODE128, PDF417, DataMatrix, and more
- **Download & Share**: Save as PNG or copy to clipboard
- **Privacy First**: No data collection, works completely offline
- **Clean Interface**: Simple, intuitive design

## Supported Barcode Types

### 2D Barcodes
- QR Code
- PDF417
- DataMatrix
- AztecCode
- MaxiCode
- Dotcode

### 1D Barcodes
- CODE128
- CODE39
- EAN13
- ITF (Interleaved 2 of 5)
- MSI
- Pharmacode
- Codabar

## How to Use

1. Click the extension icon in your Chrome toolbar
2. Choose an option:
   - **Use Current URL**: Generates QR code from the current webpage
   - **Manual Input**: Enter custom text
3. Select barcode type from dropdown (default: QR Code)
4. Click **"Generate Barcode"**
5. Download as PNG or copy to clipboard

## Use Cases

- Share webpage URLs via QR code
- Generate product barcodes for inventory
- Create event tickets with unique codes
- Encode contact information
- Convert links for mobile scanning

## Technical Details

- **Library**: bwip-js v4.1.2 (included locally)
- **Manifest**: Version 3
- **Permissions**: activeTab only
- **Size**: ~1 MB
