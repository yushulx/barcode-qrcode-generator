# Privacy Policy for Barcode & QR Code Generator

**Last Updated: October 31, 2025**

## Data Collection

This extension **does NOT collect, store, or transmit any user data**.

## What We Don't Do

- ❌ No data collection
- ❌ No tracking or analytics
- ❌ No cookies
- ❌ No personal information storage
- ❌ No data transmission to external servers
- ❌ No third-party services

## How the Extension Works

1. **Local Processing**: All barcode generation happens locally in your browser
2. **Temporary URL Access**: When you click "Use Current URL", the extension reads the current page URL to generate a barcode. This URL is processed immediately and is never stored or transmitted.
3. **User Input**: Any text you manually enter is processed locally and immediately discarded after barcode generation
4. **Offline Capability**: The extension works completely offline using locally bundled libraries

## Permissions

**activeTab**: Used only to read the current page URL when you explicitly click the "Use Current URL" button. No background monitoring or automatic data access occurs.

## Data Storage

The extension does not use:
- Local storage
- Session storage
- IndexedDB
- Cookies
- Cache (except for the extension's own files)

## Third-Party Services

This extension does not connect to any external servers or third-party services. All functionality is self-contained within the extension.

## Changes to Privacy Policy

Any changes to this privacy policy will be updated in the extension's store listing.

## Contact

For questions about this privacy policy, please contact us through the Chrome Web Store support page.

This extension complies with:
- Chrome Web Store Developer Program Policies
- User Data Privacy requirements
